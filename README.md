# CheatingBooks

Serialized marriage-in-crisis romance stories. Built with Next.js 14, Tailwind CSS, and Stripe Payment Links.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Environment Variables

Copy `.env.example` to `.env` and add your Stripe Payment Link.

## Deploy

Push to GitHub and connect to [Vercel](https://vercel.com).
